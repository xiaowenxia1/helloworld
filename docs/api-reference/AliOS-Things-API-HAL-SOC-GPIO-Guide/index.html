6-hal_gpio_enable_irq

int32_t hal_gpio_enable_irq(gpio_dev_t *gpio, gpio_irq_trigger_t trigger,
                            gpio_irq_handler_t handler, void *arg)

  • [x] Description

    Enables an interrupt trigger for an input GPIO pin. Using this function on a gpio pin which is set to output mode is undefined.

  • [x] Parameters

    IN/OUTNAMEDESC
    [in]gpiothe gpio pin which will provide the interrupt trigger
    [in]triggerthe type of trigger (rising/falling edge)
    [in]handlera function pointer to the interrupt handler
    [in]argan argument that will be passed to the interrupt handle

  • [x] Returns

    0 : on success, EIO : if an error occurred with any step
